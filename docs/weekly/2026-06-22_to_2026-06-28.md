## 3. Current Technical Findings

- The default Python 3.14 environment does not have the scientific packages needed for this project.
- The existing `SMRI` environment has the main packages needed for the next pilot step: `numpy`, `pandas`, `scipy`, `matplotlib`, `scikit-learn`, `mne`, `edfio`, `ipykernel`, and `torch`.
- `pyedflib`, `wfdb`, and `notebook` are not installed in `SMRI`, but they are not required before the first EDF check.
- The next pilot should use `SMRI` unless a later compatibility issue appears.
- The `sub-53` headband and PSG EDF files both have 5,122,048 samples, 256 Hz sampling, 20,008-second duration, and the same recorded start time.
- The PSG event table contains `stage_hum`; the headband event table contains `stage_ai` only.
- The staged event tables cover 19,980 seconds, leaving a 28-second unstaged tail relative to the EDF duration.
- The pilot has six direct REM/Wake adjacencies from PSG `stage_hum`: four REM-to-Wake and two Wake-to-REM.
